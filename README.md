1. Install Git
Open Powershell by right clicking and selecting "run as Administrator"
Download Git from https://git-scm.com/download/win
Run git --version

2. UV
uv --version
install by - https://docs.astral.sh/uv/getting-started/installation/
uv init
uv add openai

         →  created .venv folder (virtual environment)
         →  Cursor detected it automatically
